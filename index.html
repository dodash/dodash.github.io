Introducing the beta program for DevOps Dashboards, a fully customizable dashboard to help you manage your software development projects. Key features in the beta: OpenSocial container – Supports OpenSocial gadgets Multiple dashboards – Create multiple project-based dashboards Shared dashboards – Share dashboards with other team members ClearQuest integration – run queries and charts with data from a ClearQuest repository Out of the box gadgets – useful out of the box gadgets to help create useful dashboards for your team Widget store – add, remove, edit, rate, comment on widgets in a social environment Refer to http://www.cccqcommunity.com/dashboard_beta.html for details
